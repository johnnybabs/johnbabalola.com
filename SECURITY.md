# Security Policy

## Reporting a security issue

If you find a security vulnerability in this repository's infrastructure code or configuration, email baabalola@gmail.com rather than opening a public issue. Please include a description of the issue, the file or resource affected, and steps to reproduce.

## Scope

This repository provisions infrastructure for a personal portfolio site. Intentionally out of scope: application-layer vulnerabilities in the static HTML content, and volumetric denial-of-service attacks against the CloudFront distribution. Security exceptions are documented with expiry dates in `docs/security-exceptions.md`.
